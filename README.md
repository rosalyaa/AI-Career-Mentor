<h1>AI Career Mentor</h1>

An AI-powered resume analyzer built using Python and Streamlit.

<h2>Demo:</h2> 

[http://localhost:8502](http://localhost:8502)

<h2>Features:</h2>

- Resume Upload
- Resume Score
- Skill Extraction
- Career Recommendation
- Skill Gap Analysis
- Learning Roadmap

<h2>Technologies:</h2>

- Python
- Streamlit
- HTML
- CSS
- PyMuPDF
- Regex
